Junction [jcn] Documentation
============================

Command-line applications are just plain cool. You don't look like a genuine
computer geek without being able to a drive blocky app with your hands flying
over the keyboard. To that end, Junction is a Python-based command-line UI
framework, designed to make programming awesome command-line applications a
breeze.
